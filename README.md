# DabaFood

Site vitrine d'un restaurant ivoirien gastronomique basé à Abidjan.  
Design sombre et élégant, pensé pour refléter le positionnement premium de la marque.

**Live demo :** [daba-food.vercel.app](https://daba-food.vercel.app/)

---

## Aperçu

DabaFood est un site one-page responsive pour un restaurant de cuisine ivoirienne haut de gamme. Il présente la carte, l'histoire du restaurant, et permet aux clients de réserver via WhatsApp directement depuis le site.

---

## Stack

- **HTML5** — structure sémantique
- **Tailwind CSS** (CDN) — styling utilitaire
- **CSS custom** — glassmorphism, animations, scroll reveal
- **JavaScript vanilla** — interactions, navbar au scroll, formulaire WhatsApp

---

## Fonctionnalités

- Hero plein écran avec overlay élégant
- Menu des spécialités avec layout asymétrique
- Section À propos avec valeurs de la maison
- Formulaire de réservation → ouverture WhatsApp avec message pré-rempli
- Navbar fixe avec effet glassmorphism au scroll
- Scroll reveal sur les sections
- Entièrement responsive (mobile-first)

## Lancer le projet

Aucune dépendance à installer. Ouvre simplement `index.html` dans un navigateur.

```bash
git clone https://github.com/Mourtada-002/DabaFood.git
cd DabaFood
# Ouvrir index.html dans le navigateur
```

---

## Déploiement

Le site est déployé sur **Vercel** via connexion GitHub.  
Chaque push sur `main` déclenche un redéploiement automatique.

---

## Développement

Site initialement généré avec assistance IA, puis retravaillé et redesigné avec **Antigravity** pour obtenir un rendu modern et premium.
