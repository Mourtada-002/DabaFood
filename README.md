# DabaFood

A **gastronomic Ivorian restaurant showcase website** built with HTML, Tailwind CSS and vanilla JavaScript, featuring an elegant dark design crafted to reflect the brand's premium positioning.

## Features

**Fullscreen Hero** – Immersive landing section with a sleek overlay and strong visual identity  
**Asymmetric Menu Layout** – Specialty dishes presented with a distinctive, high-end editorial layout  
**WhatsApp Reservation Form** – Booking form that opens WhatsApp with a pre-filled message for seamless customer experience  
**Glassmorphism Navbar** – Fixed navigation bar with scroll-triggered glass effect for a modern, layered feel  
**Scroll Reveal Animations** – Sections animate into view as the user scrolls for a fluid, engaging experience  
**Mobile-First Responsive** – Fully adaptive layout optimized for all screen sizes  

## Technologies Used

- **HTML5** – Single-file semantic structure for lightweight delivery
- **Tailwind CSS** – Utility-first styling via CDN for rapid, consistent layouts
- **Custom CSS** – Glassmorphism effects, animations and scroll reveal
- **Vanilla JavaScript** – Scroll interactions, navbar behavior and WhatsApp form logic
